Welcome to Ticket-Show Application

Thank you for visiting my application.

  • I am delighted to have the opportunity to showcase my ticket booking application, which I've been working on after completing the App Dev 1 course. This application is designed to cater to the needs of users who want to book tickets online in a hassle-free manner.
  • It is a multi-user app (one admin & multiple users). User can book multiple shows from multiple venues while an Admin can create venues & shows. Each venue can run a number of shows.
  • I have used Flask for application code, Jinja2 templates + Bootstrap for HTML generation and styling, SQLite for data storage.
  • I hope you'll find my application impressive, and I welcome any feedback or suggestions you may have.
